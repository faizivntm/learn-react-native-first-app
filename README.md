# 📱 React Native Learning Collection

Repositori ini berisi kumpulan materi dasar dalam pengembangan aplikasi menggunakan React Native.  
Cocok untuk pemula yang ingin memahami konsep fundamental sebelum membangun aplikasi yang lebih kompleks.

---

## 🚀 Materi yang Dipelajari

### 🧩 Component
Memahami cara membuat dan menggunakan komponen sebagai struktur utama dalam React Native.

### 🔗 Props
Mengirim data antar komponen agar aplikasi lebih dinamis dan reusable.

### 🧠 State
Mengelola data dalam komponen serta memahami bagaimana perubahan state mempengaruhi UI.

### ⚡ Event Handler
Menangani interaksi pengguna seperti klik, input, dan aksi lainnya.

### 🔀 Conditional Rendering
Menampilkan UI berdasarkan kondisi tertentu menggunakan `if` atau ternary operator.

---

## 🎯 Tujuan Proyek

- Membantu memahami konsep dasar React Native
- Menjadi referensi belajar yang sederhana dan mudah dipahami
- Menyediakan contoh implementasi langsung dari tiap konsep
